# API Documentation

This document provides detailed information about the custom API endpoints in Travel Genie.

See the main README.md Developer Manual section for full API documentation.

## Quick Reference

- **GET/POST `/api/trips`** - Manage user trips
- **GET/POST `/api/search-history`** - Manage and enrich search history

For detailed endpoint documentation, parameters, and examples, refer to the [Developer Manual API Documentation](../README.md#api-documentation) section in README.md.
